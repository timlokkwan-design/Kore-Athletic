# Kore Athletic 綜合更新（2026.07.19-201）

本版本合併 PR #1–#15 功能，並**永久保障學生用戶不會因更新而消失**。

- 所有 `users` 寫入經 `protected_save_users`：merge-protect，唔會整表清空真實學員
- 獨立 users 備份；教練可於「網站內容」還原
- Forever rule：`.cursor/rules/protect-student-users.mdc`

## 程式庫清理（B1）
- 已刪除根目錄與 `utils/`／`views/` **重複的 46 個 `.py` 死碼**
- 正式 App 只認 `app.py` → `utils.*`／`views.*`
- 根目錄只保留入口／維運腳本：`app.py`、`migrate_*.py`、`preflight.py`、`reset_*.py` 等
- 一併消除根目錄 `calendar.py` 遮蔽 Python 標準庫 `calendar` 的問題

1. 登入狀態持久／教練待審彈窗／預早請假
2. 手機待審橫幅
3. 學生帳號保護＋備份
4. Instagram 式底部導航（學生＋教練）

8. 設定課表 ↔ 訓練時間表日曆同步
9. PB 排行榜：登入可見姓名
10. 學生訓練日誌：記錄／過往／課表比較
11. 學生主頁訂立目標

13. 最新消息（教練發佈／學生可睇）
14. 比賽成功報名名單（含分齡賽五預填）
